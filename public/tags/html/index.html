常用的一些html小问题

1. 网页出现横向滚动条？ <body style=`overflow:-Scroll;overflow-y:hidden > </body> 让横条没有： <body style=`overflow:-Scroll;overflow-x:hidden> </body> 两个都去掉？更简单了 <body scroll="no" > </body> 火狐底部滚动条不显示： html { overflow:-moz-scrollbars-vertical; } 2. 网页图标？ <link rel="shortcut icon " href="images/favicon.ico"> 3. Height:100%失效？ <html style="height: 100%;"> <body style="height: 100%;"> <div style="height: 100%;"> <p> 这样这个div的高度就会100%了 </p> </div> </body> </html> 4. 设置一个元素不可见？ opacity 设为 0、将 visibility 设为 hidden、将 display 设为 none 或者将 position 设为 absolute 注意： 1> display=none元素消失，不占位置 2> opacity =0、 visibility =hidden只是视觉上不可见，其实还在那里占位置 5.input无法输入？ ==> height:0; (将包裹提示的div高度设置为0，让其不再遮挡即可)...
